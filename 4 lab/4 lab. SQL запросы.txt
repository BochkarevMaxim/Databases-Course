Вариант 3: Лекарства, больные, назначения.

Необходимо в Microsoft Access создать базу данных предметной области, 
которая соответсвует вашему варианту.
БД должна соответсвовать 3нф. Необходимо заполнить таблицы данными. 
По 5 записей минимум в каждой таблице.
Необходимо написать/создать sql запросы:

1. INSERT

INSERT INTO Врач
VALUES ("300556170", "Соколов Иван Сергеевич", "Окулист", "7", "89201272151");

2. DELETE

DELETE Врач.[Номер лицензии врача]
FROM Врач
WHERE (((Врач.[Номер лицензии врача])=300556170));

3. UPDATE

UPDATE Врач 
SET Врач.ФИО = "Скворцова Анастасия Викторовна"
WHERE (((Врач.ФИО)="Быстрыгова Анастасия Викторовна"));

4. SELECT в рамках одной таблицы

	4.1. С определенным набором извлекаемых атрибутов (SELECT atr1, atr2 FROM...)
	
	SELECT ФИО, Специальность, [Номер кабинета]
	FROM Врач
	ORDER BY ФИО;

	4.2. Со всеми атрибутами (SELECT * FROM...)

	SELECT *
	FROM Врач;

	4.3. С условием по атрибуту (SELECT * FROM ... WHERE atr1 = "")

	SELECT *
	FROM Врач
	WHERE Специальность = "Педиатр";
